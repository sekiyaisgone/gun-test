AWP Icon Assets - Required Image Files
=======================================

Place the following .png image files in this directory:

1. Weapon_AWP.png
   - Inventory icon for the AWP Sniper Rifle
   - Size: 64x64 pixels
   - Format: PNG with transparency
   - Should show a side profile of the AWP rifle
   - Tip: Render from the 3D model in Blockbench using
     the icon export feature with these settings:
       Scale: 0.28
       Rotation: [342, 56, 6]
       Translation: [-8, -32]

2. AWP_Ammo_Box.png
   - Inventory icon for AWP ammunition
   - Size: 64x64 pixels
   - Format: PNG with transparency
   - Should show an ammo box or bullet rounds
   - Tip: Render from the ammo box model with these settings:
       Scale: 0.85
       Rotation: [180, 67, 129]
       Translation: [5, -2]

Generation method:
- Open the .blockymodel in Blockbench
- Use File > Export > Item Icon
- Set the resolution to 64x64
- Adjust rotation/scale per the settings above
