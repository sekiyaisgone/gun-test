AWP Ammo Assets - Required Files
==================================

Place the following files in this directory:

1. AWP_Ammo.blockymodel
   - 3D model for the ammo box/rounds
   - Create in Blockbench with Hytale plugin
   - Format: hytale_prop
   - Simple box shape with bullet details
   - Resolution: 32x32 or 64x64 texture

2. AWP_Ammo_Texture.png
   - Texture for the ammo box model
   - Size: Match model resolution (32x32 or 64x64)
   - Color scheme: Brass/gold bullets, dark cardboard box

Design suggestions:
- Small rectangular ammo box (roughly 6x4x3 blocks)
- Visible bullet tips poking out from top
- Label/markings on the box side
- Brass colored rounds, dark green/brown box
